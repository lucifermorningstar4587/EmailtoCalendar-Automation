## Prerequisites

Before running this project, make sure you have:
- An Android device
- A Google account with Google Calendar access
- MobileRun installed on your system
- DroidRun APK installed on the Android device
- Internet connection on both mobile and system

---

## Installation

1. Clone the repository:
   git clone https://github.com/lucifermorningstar4587/EmailtoCalendar-Automation.git

2. Install the DroidRun APK on your Android device.

3. Connect the Android device to MobileRun.

4. Ensure Gmail and Google Calendar permissions are enabled on the device.

---

## Usage

1. Open MobileRun and load the automation workflow.
2. Provide the input date from which emails should be scanned.
3. Start the automation.
4. The system scans TPC emails after the given date.
5. Detected deadlines are automatically added to Google Calendar.
6. A reminder is set 30 minutes before each deadline.

The automation runs in the background once started.
